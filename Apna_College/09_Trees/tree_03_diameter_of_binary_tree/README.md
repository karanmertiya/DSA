# Tree 03 Diameter Of Binary Tree

## Description
Given the root of a binary tree, return the length of the diameter of the tree. The diameter is the length of the longest path between any two nodes.<br><br><b>🔗 Practice:</b> <a href='https://leetcode.com/problems/diameter-of-binary-tree/' target='_blank'>LeetCode 543</a>

## Examples
<b>Example 1:</b> Input: root = [1,2,3,4,5], Output: 3
