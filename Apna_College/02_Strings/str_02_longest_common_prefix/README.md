# Str 02 Longest Common Prefix

## Description
Write a function to find the longest common prefix string amongst an array of strings. If there is no common prefix, return an empty string ''.<br><br><b>🔗 Practice:</b> <a href='https://leetcode.com/problems/longest-common-prefix/' target='_blank'>LeetCode 14</a>

## Examples
<b>Example 1:</b> Input: strs = ['flower','flow','flight'], Output: 'fl'
