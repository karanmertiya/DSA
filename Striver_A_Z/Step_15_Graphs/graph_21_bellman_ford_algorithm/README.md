# Graph 21 Bellman Ford Algorithm

## Description
Given a weighted and directed graph of V vertices and E edges, Find the shortest distance of all the vertex's from the source vertex S. If a negative weight cycle is present, return an array consisting of only -1.<br><br><b>🔗 Practice:</b> <a href='https://practice.geeksforgeeks.org/problems/distance-from-the-source-bellman-ford-algorithm/0' target='_blank'>GFG</a>
