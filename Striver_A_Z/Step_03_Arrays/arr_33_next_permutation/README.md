# Arr 33 Next Permutation

## Description
Implement next permutation, which rearranges numbers into the lexicographically next greater permutation of numbers. If such an arrangement is not possible, it must rearrange it as the lowest possible order (i.e., sorted in ascending order).<br><br><b>🔗 Practice:</b> <a href='https://leetcode.com/problems/next-permutation/' target='_blank'>LeetCode 31</a>

## Examples
<b>Example 1:</b> Input: nums = [1,2,3], Output: [1,3,2]
