# Arr 60 Merge Intervals

## Description
Given an array of intervals where intervals[i] = [starti, endi], merge all overlapping intervals, and return an array of the non-overlapping intervals that cover all the intervals in the input.<br><br><b>🔗 Practice:</b> <a href='https://leetcode.com/problems/merge-intervals/' target='_blank'>LeetCode 56</a>
