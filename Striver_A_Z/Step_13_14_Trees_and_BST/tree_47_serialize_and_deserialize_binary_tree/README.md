# Tree 47 Serialize And Deserialize Binary Tree

## Description
Serialization is the process of converting a data structure or object into a sequence of bits so that it can be stored in a file or memory buffer, or transmitted across a network connection link to be reconstructed later in the same or another computer environment. Design an algorithm to serialize and deserialize a binary tree.<br><br><b>🔗 Practice:</b> <a href='https://leetcode.com/problems/serialize-and-deserialize-binary-tree/' target='_blank'>LeetCode 297</a>

## Examples
<b>Example 1:</b> Level-order traversal (BFS) with a queue.
