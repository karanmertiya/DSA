# Heap 19 Sort Characters By Frequency

## Description
Given a string s, sort it in decreasing order based on the frequency of the characters. The frequency of a character is the number of times it appears in the string. Return the sorted string. If there are multiple answers, return any of them.<br><br><b>🔗 Practice:</b> <a href='https://leetcode.com/problems/sort-characters-by-frequency/' target='_blank'>LeetCode 451</a>

## Examples
<b>Example 1:</b> Hash map + Max-heap or Sort.
