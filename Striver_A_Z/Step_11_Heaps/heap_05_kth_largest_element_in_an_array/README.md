# Heap 05 Kth Largest Element In An Array

## Description
Given an integer array nums and an integer k, return the kth largest element in the array.<br><br><b>🔗 Practice:</b> <a href='https://leetcode.com/problems/kth-largest-element-in-an-array/' target='_blank'>LeetCode 215</a>

## Examples
<b>Example 1:</b> Input: nums = [3,2,1,5,6,4], k = 2. Output: 5
