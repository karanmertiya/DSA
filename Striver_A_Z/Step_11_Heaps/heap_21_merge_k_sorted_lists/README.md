# Heap 21 Merge K Sorted Lists

## Description
You are given an array of k linked-lists lists, each linked-list is sorted in ascending order. Merge all the linked-lists into one sorted linked-list and return it.<br><br><b>🔗 Practice:</b> <a href='https://leetcode.com/problems/merge-k-sorted-lists/' target='_blank'>LeetCode 23</a>

## Examples
<b>Example 1:</b> Min-heap of list nodes.
