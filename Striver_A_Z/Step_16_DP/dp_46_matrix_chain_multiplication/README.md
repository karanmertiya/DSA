# Dp 46 Matrix Chain Multiplication

## Description
Given a sequence of matrices, find the most efficient way to multiply these matrices together. The problem is not actually to perform the multiplications, but merely to decide in which order to perform the multiplications. The matrices have sizes arr[i-1] x arr[i].<br><br><b>🔗 Practice:</b> <a href='https://practice.geeksforgeeks.org/problems/matrix-chain-multiplication0303/1' target='_blank'>GFG</a>

## Examples
<b>Example 1:</b> Interval DP.
