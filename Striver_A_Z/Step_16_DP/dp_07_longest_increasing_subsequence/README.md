# Dp 07 Longest Increasing Subsequence

## Description
Given an integer array nums, return the length of the longest strictly increasing subsequence.<br><br><b>🔗 Practice:</b> <a href='https://leetcode.com/problems/longest-increasing-subsequence/' target='_blank'>LeetCode 300</a>

## Examples
<b>Example 1:</b> Input: nums = [10,9,2,5,3,7,101,18], Output: 4 (LIS is [2,3,7,101])
