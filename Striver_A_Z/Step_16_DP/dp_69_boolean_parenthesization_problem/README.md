# Dp 69 Boolean Parenthesization Problem

## Description
Given a boolean expression S of length N with following symbols. Symbols 'T' ---> true, 'F' ---> false and operators &, |, ^. Count the number of ways we can parenthesize the expression so that the value of expression evaluates to true.<br><br><b>🔗 Practice:</b> <a href='https://practice.geeksforgeeks.org/problems/boolean-parenthesization5610/1' target='_blank'>GFG</a>
