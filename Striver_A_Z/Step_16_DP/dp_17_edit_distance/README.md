# Dp 17 Edit Distance

## Description
Given two strings word1 and word2, return the minimum number of operations required to convert word1 to word2. Operations are insert, delete, or replace a character.<br><br><b>🔗 Practice:</b> <a href='https://leetcode.com/problems/edit-distance/' target='_blank'>LeetCode 72</a>

## Examples
<b>Example 1:</b> Input: word1 = 'horse', word2 = 'ros', Output: 3
