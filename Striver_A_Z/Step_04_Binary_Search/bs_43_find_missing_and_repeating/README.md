# Bs 43 Find Missing And Repeating

## Description
Given an unsorted array Arr of size N of positive integers. One number 'A' from set {1, 2, …,N} is missing and one number 'B' occurs twice in array. Find these two numbers.<br><br><b>🔗 Practice:</b> <a href='https://practice.geeksforgeeks.org/problems/find-missing-and-repeating2512/1' target='_blank'>GFG</a>

## Examples
<b>Example 1:</b> Math or XOR.
