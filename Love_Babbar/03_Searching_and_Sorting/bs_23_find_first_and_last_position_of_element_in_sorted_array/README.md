# Bs 23 Find First And Last Position Of Element In Sorted Array

## Description
Given an array of integers nums sorted in non-decreasing order, find the starting and ending position of a given target value.<br><br><b>🔗 Practice:</b> <a href='https://leetcode.com/problems/find-first-and-last-position-of-element-in-sorted-array/' target='_blank'>LeetCode 34</a>

## Examples
<b>Example 1:</b> Input: nums = [5,7,7,8,8,10], target = 8, Output: [3,4]
