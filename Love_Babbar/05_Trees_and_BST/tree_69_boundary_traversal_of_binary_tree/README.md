# Tree 69 Boundary Traversal Of Binary Tree

## Description
Given a Binary Tree, find its Boundary Traversal. The traversal should be in the following order: 1. Left boundary nodes 2. Leaf nodes 3. Right boundary nodes in reverse.<br><br><b>🔗 Practice:</b> <a href='https://practice.geeksforgeeks.org/problems/boundary-traversal-of-binary-tree/1' target='_blank'>GFG</a>

## Examples
<b>Example 1:</b> Left boundary, then leaves, then right boundary.
