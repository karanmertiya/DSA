# Tree 53 Insert Into A Binary Search Tree

## Description
You are given the root node of a binary search tree (BST) and a value to insert into the tree. Return the root node of the BST after the insertion. It is guaranteed that the new value does not exist in the original BST.<br><br><b>🔗 Practice:</b> <a href='https://leetcode.com/problems/insert-into-a-binary-search-tree/' target='_blank'>LeetCode 701</a>

## Examples
<b>Example 1:</b> Iterative search and insert.
