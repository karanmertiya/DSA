# Sw 63 Minimum Size Subarray Sum

## Description
Given an array of positive integers nums and a positive integer target, return the minimal length of a contiguous subarray [numsl, numsl+1, ..., numsr-1, numsr] of which the sum is greater than or equal to target. If there is no such subarray, return 0 instead.<br><br><b>🔗 Practice:</b> <a href='https://leetcode.com/problems/minimum-size-subarray-sum/' target='_blank'>LeetCode 209</a>

<b>Example 1:</b> Sliding Window.
