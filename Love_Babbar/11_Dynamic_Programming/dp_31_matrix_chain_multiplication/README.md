# Dp 31 Matrix Chain Multiplication

## Description
Given a sequence of matrices, find the most efficient way to multiply these matrices together. The efficient way is the one that involves the least number of multiplications.<br><br><b>🔗 Practice:</b> <a href='https://practice.geeksforgeeks.org/problems/matrix-chain-multiplication0303/1' target='_blank'>GFG</a>

## Examples
<b>Example 1:</b> Minimize scalar multiplications.
