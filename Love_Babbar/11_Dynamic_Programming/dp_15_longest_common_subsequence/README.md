# Dp 15 Longest Common Subsequence

## Description
Given two strings text1 and text2, return the length of their longest common subsequence. If there is no common subsequence, return 0.<br><br><b>🔗 Practice:</b> <a href='https://leetcode.com/problems/longest-common-subsequence/' target='_blank'>LeetCode 1143</a>

## Examples
<b>Example 1:</b> Input: text1 = 'abcde', text2 = 'ace', Output: 3
