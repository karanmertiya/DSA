# Heap 19 Find Median From Data Stream

## Description
The median is the middle value in an ordered integer list. If the size of the list is even, there is no middle value, and the median is the mean of the two middle values. Implement the MedianFinder class to add numbers from a data stream and find the median.<br><br><b>🔗 Practice:</b> <a href='https://leetcode.com/problems/find-median-from-data-stream/' target='_blank'>LeetCode 295</a>

## Examples
<b>Example 1:</b> Two heaps (Max-heap for lower half, Min-heap for upper half).
