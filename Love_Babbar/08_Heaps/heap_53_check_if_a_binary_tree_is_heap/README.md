# Heap 53 Check If A Binary Tree Is Heap

## Description
Given a binary tree. The task is to check whether the given tree follows the max heap property or not. Note: Properties of a tree to be a max heap - Completeness and Value of node greater than or equal to its child.<br><br><b>🔗 Practice:</b> <a href='https://practice.geeksforgeeks.org/problems/is-binary-tree-heap/1' target='_blank'>GFG</a>
