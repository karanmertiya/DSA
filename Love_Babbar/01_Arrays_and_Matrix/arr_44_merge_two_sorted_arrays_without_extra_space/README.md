# Arr 44 Merge Two Sorted Arrays Without Extra Space

## Description
Given two sorted arrays arr1[] and arr2[] of sizes n and m in non-decreasing order. Merge them in sorted order without using any extra space. Modify arr1 so that it contains the first n elements and modify arr2 so that it contains the last m elements.<br><br><b>🔗 Practice:</b> <a href='https://practice.geeksforgeeks.org/problems/merge-two-sorted-arrays-1587115620/1' target='_blank'>GFG</a>

## Examples
<b>Example 1:</b> Gap method (Shell sort idea).
