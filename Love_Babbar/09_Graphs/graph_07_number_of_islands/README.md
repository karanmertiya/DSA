# Graph 07 Number Of Islands

## Description
Given an m x n 2D binary grid grid which represents a map of '1's (land) and '0's (water), return the number of islands.<br><br><b>🔗 Practice:</b> <a href='https://leetcode.com/problems/number-of-islands/' target='_blank'>LeetCode 200</a>

## Examples
<b>Example 1:</b> Input: grid=[['1','1','0','0','0'],...], Output: 1
