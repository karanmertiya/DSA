# Tree 91 Binary Tree Maximum Path Sum

## Description
A path in a binary tree is a sequence of nodes where each pair of adjacent nodes in the sequence has an edge connecting them. Given the root of a binary tree, return the maximum path sum of any non-empty path.<br><br><b>🔗 Practice:</b> <a href='https://leetcode.com/problems/binary-tree-maximum-path-sum/' target='_blank'>LeetCode 124</a>

## Examples
<b>Example 1:</b> Postorder Traversal.
