# Tree 67 Check If A Binary Tree Is Balanced

## Description
Given a binary tree, determine if it is height-balanced. A height-balanced binary tree is defined as a binary tree in which the left and right subtrees of every node differ in height by no more than 1.<br><br><b>🔗 Practice:</b> <a href='https://leetcode.com/problems/balanced-binary-tree/' target='_blank'>LeetCode 110</a>

## Examples
<b>Example 1:</b> DFS post-order.
