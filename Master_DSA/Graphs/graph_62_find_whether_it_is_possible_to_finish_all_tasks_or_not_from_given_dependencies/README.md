# Graph 62 Find Whether It Is Possible To Finish All Tasks Or Not From Given Dependencies

## Description
There are a total of n tasks you have to pick, labeled from 0 to n-1. Some tasks may have prerequisites, for example to pick task 0 you have to first pick task 1, which is expressed as a pair: [0, 1]. Given the total number of tasks and a list of prerequisite pairs, is it possible for you to finish all tasks?<br><br><b>🔗 Practice:</b> <a href='https://leetcode.com/problems/course-schedule/' target='_blank'>LeetCode 207</a>

## Examples
<b>Example 1:</b> Detect cycle using Kahn's algorithm.
