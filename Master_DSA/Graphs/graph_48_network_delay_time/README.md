# Graph 48 Network Delay Time

## Description
You are given a network of n nodes, labeled from 1 to n. You are also given times, a list of travel times as directed edges times[i] = (ui, vi, wi), where ui is the source node, vi is the target node, and wi is the time it takes for a signal to travel from source to target. We will send a signal from a given node k. Return the time it takes for all the n nodes to receive the signal. If it is impossible for all the n nodes to receive the signal, return -1.<br><br><b>🔗 Practice:</b> <a href='https://leetcode.com/problems/network-delay-time/' target='_blank'>LeetCode 743</a>

## Examples
<b>Example 1:</b> Dijkstra's Algorithm.
