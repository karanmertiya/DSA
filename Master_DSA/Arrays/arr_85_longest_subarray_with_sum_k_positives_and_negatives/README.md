# Arr 85 Longest Subarray With Sum K Positives And Negatives

## Description
Given an array containing N integers and an integer K, Your task is to find the length of the longest Sub-Array with the sum of the elements equal to the given value K. The array can contain both positive and negative integers.<br><br><b>🔗 Practice:</b> <a href='https://practice.geeksforgeeks.org/problems/longest-sub-array-with-sum-k0809/1' target='_blank'>GFG</a>

## Examples
<b>Example 1:</b> Hash Map.
