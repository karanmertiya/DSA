# Bs 17 Koko Eating Bananas

## Description
Koko loves to eat bananas. Return the minimum integer k such that she can eat all the bananas within h hours.<br><br><b>🔗 Practice:</b> <a href='https://leetcode.com/problems/koko-eating-bananas/' target='_blank'>LeetCode 875</a>

## Examples
<b>Example 1:</b> Input: piles = [3,6,7,11], h = 8, Output: 4
