# Hash 17 4Sum

## Description
Given an array nums of n integers, return an array of all the unique quadruplets [nums[a], nums[b], nums[c], nums[d]] such that a, b, c, and d are distinct, and nums[a] + nums[b] + nums[c] + nums[d] == target.<br><br><b>🔗 Practice:</b> <a href='https://leetcode.com/problems/4sum/' target='_blank'>LeetCode 18</a>
