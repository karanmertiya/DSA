# Str 14 Valid Anagram

## Description
Given two strings s and t, return true if t is an anagram of s, and false otherwise. An Anagram is a word or phrase formed by rearranging the letters of a different word or phrase, typically using all the original letters exactly once.<br><br><b>🔗 Practice:</b> <a href='https://leetcode.com/problems/valid-anagram/' target='_blank'>LeetCode 242</a>
